## 🚀 Running the Notebooks

1. Clone the repository
```bash
git clone https://github.com/Raj-Kori/PRODIGY_ML.git
cd PRODIGY_ML
```

2. Install dependencies
```bash
pip install numpy pandas scikit-learn tensorflow matplotlib jupyter
```

3. Launch Jupyter
```bash
jupyter notebook
```

4. Open any `.ipynb` file in your browser
